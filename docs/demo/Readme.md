## TSL/SSL 证书
生成非对称密钥不是目的，而是为了其它目的服务。

1. 生成证书请求文件 .csr
  - 已有私钥
  - 无私钥
2. 生成自签证书 .crt
  - 已有私钥和 .csr
  - 已有私钥，无 .csr
  - 无私钥，无 .csr
3. 生成自签 CA 证书
4. 签发证书
  - 已有 csr
5. 二级 CA 证书
  - 生成二级 CA 证书的请求文件 .csr
  - 签发二级 CA 证书
6. 根 CA 证书到期后的更换：key 不变，证书换新（有效期更新）

以上，背诵对应配置文件
